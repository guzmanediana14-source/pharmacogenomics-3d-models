# Integración clínico-genómica de las variantes en CYP3A5 Y CYP2C19 en toxicidad serotoninérgica asociada a sertralina en monoterapia

## Descripción general
Este repositorio contiene los modelos estructurales utilizados y generados durante el estudio de variantes farmacogenómicas en los genes **CYP2C19** y **CYP3A5**. Los archivos fueron empleados para análisis estructurales, comparación conformacional y visualización molecular.

## Contenido del repositorio

### 01_CYP3A5_WT
Modelo estructural de referencia de CYP3A5. Incluye el grupo hemo y representa la conformación funcional completa de la proteína.

### 02_CYP3A5_Truncada
Modelo estructural truncado de CYP3A5 generado para explorar las posibles consecuencias estructurales de alteraciones en el procesamiento del ARN mensajero asociadas a variantes genéticas. El modelo carece de la región C-terminal necesaria para la formación del sitio de unión al grupo hemo, por lo tanto, el modelo carece del mismo.

### 03_CYP2C19_WT
Modelo estructural de referencia de CYP2C19 utilizado para el análisis de interacción molecular. Incluye el grupo hemo característico de las enzimas del citocromo P450.

### 04_CYP2C19_I331V
Modelo estructural de la variante CYP2C19-I331V generado mediante modelado estructural. Este modelo fue utilizado para evaluar posibles cambios conformacionales asociados a la variante genética y su potencial impacto sobre la interacción con fármacos.
